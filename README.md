# 九万亩风格战略占地 Demo（前端原型）

> 说明：本项目是“玩法向”原型，目标是还原 **主城开局 → 边界扩张 → 荒地争夺 → 小格战斗** 的战略循环。

## 当前玩法

- 世界地图：`600 x 600`
- 对局规模：`500` 名玩家（本地 AI 模拟）
- 开局规则：每名玩家仅 1 座主城
- 扩张规则：只能从接壤边界向荒地推进
- 战斗规则：每个目标格子触发一场 `9 x 9` 回合制自走棋战斗
- 行军机制：点击荒地后先“行军”，到达后再开战

## 运行

```bash
python3 -m http.server 8000
```

浏览器打开：`http://localhost:8000`

## 操作

1. 在世界地图中点击与你领土相邻的荒地。
2. 观察行军数量变化（顶部“行军”）。
3. 行军抵达后右侧自动进入 9x9 战斗。
4. 点击“下一回合”或“自动10回合”推进。
5. 攻方胜利则该格子归你所有。

## 结构

- `index.html`：UI结构
- `styles.css`：界面样式
- `game.js`：世界模拟、行军、战斗逻辑
