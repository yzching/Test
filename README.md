# 九万亩风格战略游戏 Demo（可玩版本）

这版不是资料页，而是可玩对局原型：

- 600x600 世界地图
- 500 人同图（本地AI模拟）
- 主城开局 + 边界扩张
- 点击荒地发起行军
- 抵达后进入 9x9 回合制战斗
- 占领 20 块领土获胜

## 运行

```bash
python3 -m http.server 8000
```

打开：`http://localhost:8000`

## 操作

1. 点击“开始对局”。
2. 点击你领土边界外的灰色荒地，派遣行军。
3. 行军到达后，右侧会自动生成 9x9 战斗。
4. 点击“下一回合”或“自动10回合”推进战斗。
5. 方向键可移动世界视野。

## 胜负

- 胜利：领土达到 20。
- 失败：兵力耗尽且无行军、无战斗进行中。
