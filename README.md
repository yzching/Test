# 600x600 战略占地 Demo（类九万亩题材玩法）

这个版本重点改为你要求的**战略玩法**：

- 一局支持 500 名玩家（本地模拟 AI）
- 每名玩家初始 1 座主城
- 通过士兵攻打“荒地”扩张领土
- 总地图规模 600 x 600
- 每个被攻击格子会进入 9x9 回合制自走旗战斗

> 说明：这是纯前端原型，500 人为“本地模拟对局”而非联网实时服务器。

## 运行

```bash
python3 -m http.server 8000
```

打开：`http://localhost:8000`

## 操作说明

1. 蓝色区域是你的领土，黄色小块是主城。
2. 点击与你领土相邻的灰色荒地发起进攻。
3. 右侧 9x9 战斗盘会生成战斗。
4. 点击“执行下一回合”推进战斗。
5. 攻方获胜后占领该格子并增加领土。

## 文件

- `index.html` 页面结构
- `styles.css` UI 样式
- `game.js` 世界地图、AI扩张、9x9战斗逻辑
